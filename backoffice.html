🚜 Back Office - Gestion des Manifestations

📋 Instructions

Format Excel requis :

  • lieu : Nom de la ville/lieu
  • dept : Numéro du département
  • date : Date lisible (ex: "5 mai 2026")
  • dateSort : Date au format YYYY-MM-DD (ex: "2026-05-05")
  • type : Type d'action (peut être n'importe quel texte)
  • syndicat : Nom du syndicat
  • participants : Nombre de participants
  • details : Description de la manifestation
  • lat : Latitude (décimal)
  • lon : Longitude (décimal)
  • couleur : Code couleur hexadécimal (ex: "#e74c3c") - Optionnel, sera généré aléatoirement si absent

Note : Vous pouvez créer autant de types d'action que vous le souhaitez. La légende et les filtres s'adapteront automatiquement !
