📋 Instructions d'utilisation

  1. Téléchargez le modèle Excel ci-dessus
  2. Ouvrez-le avec Excel, LibreOffice ou Google Sheets
  3. Remplissez vos données (vous pouvez supprimer les exemples)
  4. Sauvegardez le fichier au format .xlsx
  5. Importez-le ci-dessous pour le géocoder et publier

Colonnes du fichier :

  • lieu : Nom de la ville/lieu
  • dept : Numéro du département
  • date : Date lisible (ex: "5 mai 2026")
  • dateSort : Date au format YYYY-MM-DD (ex: "2026-05-05")
  • type : Type d'action (libre, créez vos propres catégories !)
  • syndicat : Nom du syndicat
  • participants : Nombre de participants
  • details : Description de la manifestation
  • lat : Latitude (laissez vide, sera calculée automatiquement)
  • lon : Longitude (laissez vide, sera calculée automatiquement)
  • couleur : Code couleur hexadécimal (ex: "#e74c3c") - Optionnel

💡 Astuce : Vous pouvez créer autant de types d'action que vous le souhaitez. La carte s'adaptera automatiquement !
